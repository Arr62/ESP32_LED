# ESP32_LED
This project was to create a LED controller using ESP32 WROOM-32D MCU that consists of modes:
- pulse
- steady

Used elements:
- bipolar transistor NPN BC547c
- 1x resistor 2.5 [kΩ], 1x resistor 10 [kΩ]
- 1x capacitor 100 [uF]
